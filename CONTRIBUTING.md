# Contributing to Image Steganography Studio

Thank you for helping improve this project.

## Ways to contribute

- Report bugs.
- Suggest UX improvements.
- Improve documentation.
- Add tests.
- Refactor or optimize the codebase.
- Improve accessibility and usability of the GUI.

## Development setup

1. Fork the repository.
2. Create a branch from `main`.
3. Create a virtual environment.
4. Install dependencies from [requirements.txt](requirements.txt).
5. Make changes.
6. Run a basic validation step.
7. Open a pull request.

## Branch naming

Examples:

- `feature/add-export-warning`
- `fix/decode-validation`
- `docs/readme-screenshots`

## Pull request checklist

- Keep changes focused.
- Update docs when behavior changes.
- Avoid unrelated formatting changes.
- Verify the app still launches.
- Add clear reproduction details for bug fixes.

## Code style

- Prefer descriptive names.
- Keep classes small and focused.
- Preserve separation between GUI and domain logic.
- Document public functions and classes when helpful.

## Good first contribution ideas

- Add automated tests.
- Add drag-and-drop image support.
- Add preview thumbnails.
- Add dark theme support.
- Add localization support.

## Reporting security issues

Please follow [SECURITY.md](SECURITY.md) for sensitive disclosures.
